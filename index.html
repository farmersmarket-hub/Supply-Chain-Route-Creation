Supply Chain Route Creation

Manager completes this form to create or amend a supply chain pickup route. Only managers have access to this form.

📖 Key Definitions

Route Run
A single scheduled trip across all vendor stops on this route (e.g., one Thursday pickup).
Aggregator
The business/driver responsible for picking up produce from all vendors on the route and delivering it to the purchaser.
Pick-up
The total of all produce collected from a single vendor in one route run.
Load
A single produce item in a single pick-up for a single vendor.
Program Period
The full duration of the supply chain program, from start date to end date.
